# RustSmallProj
Here are some small projects in the rust language to improve my rust understanding

## note_storage:
The idea of this project is to be able to read and write from disk.
I wanted to learn how to store and change content in a file, not just variables in memory.
The information is stored in a txt file that is created if it doesn't already exist in the current directory.
It was also a good excersice in match trees.
